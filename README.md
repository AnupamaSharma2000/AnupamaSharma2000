## 🛠️ Projects

### 🏠 Floor Plan Symbol Detector — Architectural Drawing CV Pipeline *(Mar 2026)*
End-to-end computer vision pipeline for detecting architectural symbols (doors, windows, rooms, walls) in floor plan PDFs.
- Fine-tuned **Faster R-CNN (MobileNetV3 FPN)** in PyTorch on 800 real floor plans from the CubiCasa5K dataset (5,000 annotated drawings).
- Designed preprocessing pipeline: PDF-to-PNG ingestion, adaptive thresholding, morphological filtering, multi-scale resize.
- Achieved **mAP@0.5 of 0.54** (P=0.577, R=0.509) vs. P=0.120/R=0.006 for the classical Hough baseline.
- Established classical OpenCV baseline using Hough Line Transform and contour analysis.
